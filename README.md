# JAEGER

A beautiful Chrome extension that discovers relevant Polymarket prediction markets for any webpage you visit. Features a clean, gold-themed interface with seamless Base wallet integration.

## Project Structure

```
MBC/
├── sidebar-src/          ← EDIT HERE (React source code)
│   ├── App.jsx
│   ├── components/
│   └── styles.css
├── sidebar/              ← BUILT OUTPUT (auto-generated, don't edit)
│   ├── index.html
│   └── assets/
├── background/           ← Background service worker
│   └── background.js
├── content/              ← Content scripts (scraping)
│   └── scrape.js
├── assets/               ← Extension icons
└── manifest.json         ← Extension config
```

**Important:** 
- Edit files in `sidebar-src/` (your React code)
- `sidebar/` is generated by `npm run build` - don't edit it directly
- Chrome extension loads from `sidebar/` folder

- ** Google-Styled UI** - Clean, modern interface with Material Design principles

## Setup

1. **Install dependencies:**
```bash
npm install
```

2. **(Optional) Add OnchainKit API Key:**
   - Get a free API key from https://portal.cdp.coinbase.com/
   - Create `.env` file in root:
   ```
   VITE_ONCHAINKIT_API_KEY=your_api_key_here
   ```

3. **Build the sidebar:**
```bash
npm run build
```

4. **Load extension in Chrome:**
   - Go to `chrome://extensions/`
   - Enable Developer Mode
   - Click "Load unpacked"
   - Select this folder

5. **Start using:**
   - Click the extension icon in Chrome's toolbar
   - Connect your wallet or create a new one
   - Browse any webpage to see relevant markets!

## Development

- Edit React code in `sidebar-src/`
- Run `npm run build` after changes
- Reload extension in Chrome
- Or use `npm run dev` for watch mode

- **Spacing**: Consistent 4px/8px/16px/24px grid
