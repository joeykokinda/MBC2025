# PolyFinder

Chrome extension that displays relevant Polymarket markets in a side panel.

## Project Structure

```
MBC/
├── sidebar-src/          ← EDIT HERE (React source code)
│   ├── App.jsx
│   ├── components/
│   └── styles.css
├── sidebar/              ← BUILT OUTPUT (auto-generated, don't edit)
│   ├── index.html
│   └── assets/
├── background/           ← Background service worker
│   └── background.js
├── content/              ← Content scripts (scraping)
│   └── scrape.js
├── assets/               ← Extension icons
└── manifest.json         ← Extension config
```

**Important:** 
- Edit files in `sidebar-src/` (your React code)
- `sidebar/` is generated by `npm run build` - don't edit it directly
- Chrome extension loads from `sidebar/` folder

## Setup

1. Install dependencies:
```bash
npm install
```

2. Build the sidebar (compiles `sidebar-src/` → `sidebar/`):
```bash
npm run build
```

3. Load extension in Chrome:
   - Go to `chrome://extensions/`
   - Enable Developer Mode
   - Click "Load unpacked"
   - Select this folder

4. Open the side panel:
   - Click the extension icon in Chrome's toolbar

## Development

- Edit React code in `sidebar-src/`
- Run `npm run build` after changes
- Reload extension in Chrome
- Or use `npm run dev` for watch mode
