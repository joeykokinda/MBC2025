# JAEGER

A beautiful Chrome extension that discovers relevant Polymarket prediction markets for any webpage you visit. Features a premium gold luxury theme with seamless Base Account wallet integration and smooth scroll-triggered animations.

## Project Structure

```
MBC2025/
├── sidebar-src/          ← EDIT HERE (React source code)
│   ├── App.jsx          ← Main React application
│   ├── components/      ← React components
│   │   ├── MarketCard.jsx
│   │   ├── FilterBar.jsx
│   │   ├── Spinner.jsx
│   │   └── StatsPanel.jsx
│   ├── styles.css       ← Main stylesheet (Gold luxury theme)
│   ├── index.html
│   └── main.jsx
├── sidebar/              ← BUILT OUTPUT (auto-generated, don't edit)
│   ├── index.html
│   └── assets/
├── background/           ← Background service worker
│   └── background.js
├── content/              ← Content scripts (scraping)
│   └── scrape.js
├── assets/               ← Extension icons & logo
│   └── jaeger_animal_128.png
├── manifest.json         ← Extension config
├── vite.config.js        ← Vite build configuration
└── package.json          ← Dependencies
```

**Important:** 
- Edit files in `sidebar-src/` (your React code)
- `sidebar/` is generated by `npm run build` - don't edit it directly
- Chrome extension loads from `sidebar/` folder
- Assets folder must be copied to build output for icons to work

## Setup

1. **Install dependencies:**
```bash
npm install
```

2. **Build the sidebar:**
```bash
npm run build
```

3. **Load extension in Chrome:**
   - Go to `chrome://extensions/`
   - Enable Developer Mode
   - Click "Load unpacked"
   - Select the `MBC2025` folder

4. **Start using:**
   - Click the JAEGER icon in Chrome's toolbar
   - Sign in with Base Account
   - Browse any webpage to discover relevant prediction markets!

## Development

- Edit React code in `sidebar-src/`
- Run `npm run build` after changes
- Reload extension in Chrome (`chrome://extensions/` → reload icon)
- Or use `npm run dev` for watch mode (auto-rebuilds on changes)

## Design System

### Color Palette - Gold Luxury Theme

**Dark Mode (Default):**
- Background: Deep OLED Black (#0a0a0a)
- Surface: Dark Gray (#141414)
- Text: Crisp White (#ffffff)
- Accent: Classic Gold (#d4af37)

**Light Mode:**
- Background: Light Gray (#f5f5f5)
- Surface: White (#ffffff)
- Text: Black (#000000)
- Accent: Classic Gold (#d4af37)

### Typography
- Font: System fonts (-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto')
- Headings: Bold (600-700), tight letter-spacing
- Body: Regular (400-500), readable line-height

### Spacing
- Consistent 8px grid system
- xs: 4px, sm: 8px, md: 16px, lg: 24px

### Animations
- Smooth cubic-bezier transitions
- Scroll-triggered market card animations
- Staggered entrance effects
- Hover interactions with gold accents

## Key Components

- **MarketCard** - Displays prediction market information with odds and volume
- **FilterBar** - Sort and filter markets (Volume, Frequency, Status)
- **Spinner** - Loading state indicator
- **StatsPanel** - Market statistics display

## Extension Configuration

- **Manifest Version**: 3
- **Permissions**: activeTab, scripting, tabs, storage, sidePanel
- **Icons**: 16px, 48px, 128px (using jaeger_animal_128.png)
- **Side Panel**: Opens in Chrome's side panel
